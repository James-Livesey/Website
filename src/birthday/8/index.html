birthday8_intro_description

2014

LiveG is founded, and its first ever software was written under the LiveG brand

The LiveG ElixOS desktop environment is created

2017

liveg.tech — the LiveG Website's principal domain name — was registered

Our first operating system prototype was created for the Raspberry Pi

2019

The gHeld was created; our first foray into the world of portable electronics

Our second generation of Adapt UI was built, alongside the Zapr tool and a new website design

2020

An early version of LiveG Sphere was developed for multiple platforms, based on Adapt UI and Electron

Our second OS prototype for the Pi was built, inspiring the interface of today's modern LiveG OS

2021

Adapt UI was overhauled, resulting in a third generation with big UI and UX improvements

Early ideas for a smartphone based on open technologies were formulated

2022

LiveG Project Prism was started, and a prototype smartphone running gShell was built

The Prism's prototype design was finalised, with gShell development advancing well

Our first episode of LiveG Showcase was released, demonstrating the Prism

gShell is ported to the desktop, making LiveG OS truly a multiplatform operating system

Our first Alpha releases of LiveG OS were made, and more gShell features were implemented

Sphere gets a tab interface to make browsing easier, and Project Proton is conceptualised

Minor projects to support LiveG's existing products were made, like Typeset Engine and Formulaic

LiveG Lookahead weekly meetings are started, bringing the community closer together

Today

Building LiveG OS for more platforms such as the PinePhone and Raspberry Pi started

More work being done to UX and configurability of LiveG OS to be more useful to future users
